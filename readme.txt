== Description ==

Displays version of WooThemes Framework in Right Now dashboard widget. If an update is available a link to the framework update page is displayed and the version that is available from WooThemes.

== Installation ==

1. Unzip the archive of the plugin.
2. Upload folder `woothemes-framework-identifier` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

Q1. Do I need to edit any files?

A1. No, just activate the plugin and it should work.
